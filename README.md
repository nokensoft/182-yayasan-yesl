# iWaTaLi Papua Website
Website profil Yayasan iWaTaLi Papua dengan tampilan landing page modern untuk menampilkan informasi organisasi, program utama, dokumentasi kegiatan, dan kanal kontak resmi.

## LIVE DEMO
https://nokensoft.github.io/176-iwatali-papua/

## Fitur Utama Website
- Hero section informatif dengan identitas iWaTaLi Papua.
- Navigasi section satu halaman (single-page) yang responsif.
- Section Tentang, Visi & Misi, Program, Wilayah Kerja, dan Kontak.
- Blog/Postingan contoh kegiatan iWaTaLi Papua.
- Galeri kegiatan interaktif dengan modal preview.
- Fitur berbagi konten galeri ke WhatsApp, Facebook, X, dan Telegram.
- Visualisasi peta sebaran program menggunakan Leaflet + OpenStreetMap.
- CTA kolaborasi dan footer informasi resmi lembaga.

## Teknologi yang Digunakan
- HTML5
- Tailwind CSS (CDN)
- Alpine.js
- Leaflet.js
- Font Awesome
- Google Fonts (Plus Jakarta Sans)
